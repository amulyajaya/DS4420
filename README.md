Predicting Flight Delays and Delay Duration

This project will analyze data on past flight delays to predict future flight delays. Flight delays cause major inconvienences for customers and airlines. By predicting future delays, airlines can have more effective and accurate schedules. This will decrease the amount of time and resources being wasted due to flight delays. There is a focus on exisiting built in delay buffers to help predict actual delay times.

The project used 2 models to analyze features and predict data. The Baysian model was implemented in python. The collaborative filtering model was implemented in R.
